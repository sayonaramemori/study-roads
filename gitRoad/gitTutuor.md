# 配置git 

## 配置用户名和邮箱

```
git config --global user.name YOUNAME
git config --global user.email YOUREMAIL
//--global is to specify the file ~/.gitconfig
//other settings include core.editor, merge.tool
```

## 查看配置信息
```
git config --list
```

## 获得帮助
```
git help <verb>
//for example: git help config
```
